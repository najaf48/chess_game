
# Chess 

A Chess game created in python using pygame.

**With an AI self playing bot.**



## Installation and Usage

Install my-project with 

```
git clone https://github.com/najaf48/chess_game.git
cd chess_game
pip install pygame
python chess.py
```
    
## Features

- Music and sound enabled 
- Vibrant colors
- Bot vs Human Mode
- Cross platform
